# Code of Conduct

Sema AI contributors are expected to work with respect, patience, and care for the communities whose languages are represented in this project.

## Our Standard

We welcome collaboration that is:

- respectful,
- constructive,
- inclusive,
- evidence-based,
- careful with sensitive language and identity data.

Unacceptable behavior includes harassment, personal attacks, discrimination, doxxing, misuse of contributor data, or attempts to bypass consent, review, privacy, or security controls.

## Reporting

Report conduct concerns privately to the project owner or a designated maintainer. Security issues should follow `SECURITY.md`.

Reports will be handled with discretion and a focus on protecting contributors and language communities.
